# 🚀 Infrastructure as Code mit Terraform, Azure & Databricks

**Beginner 101 Kurse** — Keine Vorkenntnisse nötig. Lerne Schritt für Schritt, wie du Cloud-Infrastruktur mit Code verwaltet.

---

## 📚 Kurse

| Kurs | Beschreibung | Dauer |
|------|-------------|-------|
| [🟩 Terraform 101](courses/terraform-101.html) | IaC und Terraform von Grund auf | 8 h |
| [🟦 Azure 101](courses/azure-101.html) | Azure Cloud-Grundlagen für Terraform-Nutzer | 8 h |
| [🟨 Databricks 101](courses/databricks-101.html) | Databricks-Infrastruktur und Networking | 8 h |
| [🚀 Integration 101](courses/integration-101.html) | Alle drei kombiniert: Terraform + Azure + Databricks | 10 h |

## 🎯 Was lernst du?

- **Terraform-Befehle** — `init`, `plan`, `apply` sicher beherrschen
- **HCL-Sprache** — Variablen, Outputs, Locals, Module schreiben
- **Azure-Netzwerke** — VNet, Subnet, NSG, Hub-and-Spoke verstehen
- **Databricks-Infrastruktur** — VNet Injection, SCC, Private Link
- **Best Practices** — Security Scanning, CI/CD, Remote State

## 📖 So funktionieren die Kurse

```
1. Was ist das?     → Konzepte, Analogien, "Warum?"
2. So funktioniert es → Worked Examples, Schritt-für-Schritt
3. Mach es selbst   → Geführte Labs mit Terraform-Code
4. Teste dein Wissen → Quizze mit Lösungen
5. Transferaufgabe  → Hausaufgaben, Ausblick
```

## ✅ Voraussetzungen

- Keine spezifischen Vorkenntnisse
- Empfohlen: Grundvertrautheit mit Terminal/CLI
- Benötigt: Rechner mit Editor/IDE, Internetzugang

## 📋 Reihenfolge

```
Terraform 101    → IaC, HCL, State, Module, Best Practices
       ↓
Azure 101        → RG, VNet, Subnet, NSG, Hub-and-Spoke
       ↓
Databricks 101   → Workspace, VNet Injection, SCC, NSG Rules
       ↓
Integration 101  → Alles kombiniert: Module, CI/CD, Capstone
```
